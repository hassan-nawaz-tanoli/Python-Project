# 🎓 University of Haripur – GPA & CGPA Calculator

This project is a Python-based GPA and CGPA Calculator developed by **Hassan Nawaz (Roll No: F23-1206), BS Artificial Intelligence, Department of Information Technology, University of Haripur**, under the supervision of Mam Zanaib Noor. The program calculates subject GPA based on marks according to the University grading scale, computes semester GPA using credit hours (weighted average formula), and finally calculates overall CGPA across multiple semesters. It uses Python functions, conditional statements (if-elif ladder), loops, and user input handling to perform accurate academic calculations, making it a simple yet practical academic project that demonstrates core programming concepts and real-world grading system implementation.
